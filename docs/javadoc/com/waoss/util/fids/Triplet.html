    public class Triplet<F,S,T>
extends java.lang.Object
    Represents a Pair of two arguments. A data structure capable of storing a second and third value for a first value For example, Triplet<String, Integer, Boolean> tri = new Triplet<>("Hello World", 15, true); System.out.println(tri.getFirst()); System.out.println(tri.getSecond()); System.out.println(tri.getThird());
